# FaltaUno

FaltaUno es una app web en React con experiencia tipo Tinder para descubrir eventos deportivos y unirte haciendo swipe.

## Concepto

- Swipe a la derecha: te unes al evento.
- Swipe a la izquierda: lo descartas.
- Filtros configurables: edad, nivel, distancia maxima y sexo.

## Stack

- React
- Vite
- CSS puro (responsive)

## Ejecutar en local

1. Instalar dependencias:

```bash
npm install
```

2. Iniciar entorno de desarrollo:

```bash
npm run dev
```

3. Build de produccion:

```bash
npm run build
```
